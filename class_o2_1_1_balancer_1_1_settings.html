Detailed Description

Utilities for loading configuration files for the derived programs of O2 Balancer. Since the FairRoot json loader was quite hard to adapt for custom purposes, a new library has been adapted. The library used is yaml-cpp. Using the YAML (Yaml aint a markup language) markup language.
