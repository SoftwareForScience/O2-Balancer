Introduction

The EPNs, FLPs and Information Node software all make use of the same constructs. This library contains all the abstractions and tools shared among the derived executables. Functions of the library:

  • Exception handling
  • Loading of configuration files
  • Communication with Zookeeper
  • Narrowing the usage of FairMQ
